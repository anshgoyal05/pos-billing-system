# POS Billing System

A simple **Point of Sale (POS) Billing System** built using **HTML, CSS, and JavaScript** as part of my Backend Development University coursework.

Although this project focuses on frontend technologies, it serves as the foundation for understanding how billing systems work before integrating them with backend services in future projects.

## 🚀 Features

- Add products to the cart
- Enter product price and quantity
- Display cart items in a table
- Calculate billing data using JavaScript
- Simple and responsive user interface
- Beginner-friendly project structure

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
